# GroovePuter Musical Guide (Cardputer)

This guide is aligned with the current firmware key map in `keys_sheet.md`.

## Important Cardputer Note
- No practical `Shift` is assumed.
- Use `Ctrl` or `Alt` alternatives where available.

## Fast Workflow
1. Open `Genre` and pick style/texture.
2. Open `Settings` and tune generator behavior.
3. Edit `Pattern` (`303A/303B`) and add accents/slides/rests.
4. Shape sound in `TB303 Params`.
5. Arrange in `Song` if needed.

## Musical Controls by Intent

### Groove and Style
- `Genre Page`
- `Tab` cycles `Genre -> Texture -> Presets -> Apply`.
- `Space` (on Apply focus) toggles apply scope (`SOUND+PATTERN` vs `SOUND ONLY`).
- `Enter` applies current selection; on Apply focus it toggles mode.
- `1..8` applies presets immediately and follows current apply scope.
- `M` is a quick toggle for apply scope.
- `C` toggles `CURATED` (recommended combos, marked with `*`) and `ADVANCED` (any combo).
- `G` toggles groove mode (`ACID` / `MINIMAL`).
- `0` random genre+texture.

### Generator Behavior
- `Settings Page`
- `Left/Right` adjusts selected parameter.
- `Ctrl` or `Alt` with arrows = faster stepping.
- `1..3` applies generator presets (regen scope).

### Pattern Expression
- `Pattern Edit`
- `Alt+A` accent, `Alt+S` slide.
- `R` inserts REST by clearing step.
- `Alt+Bksp` clears the full pattern.

### Sound Design
- `TB303 Params`
- `A/Z`, `S/X`, `D/C`, `F/V` for direct parameter edits.
- `T/G` oscillator, `Y/H` filter type.
- `Ctrl+Z/X/C/V` reset key synth parameters.

### Arrangement
- `Song Page`
- `Q..U` assign pattern.
- `Ctrl+Arrows` select area.
- `G` generate cell, double `G` generate row, `Alt+G` batch generate area.

For the exact and complete key list, use `keys_sheet.md`.
